# GoidaX - Браузерное расширение для обхода блокировок

Современное расширение Chrome/Firefox для разблокировки сайтов в РФ (YouTube, TikTok, AI-сервисы и др.).

## Функции
- Центральная кнопка для активации "DNS" (симуляция + инструкции)
- Лимит сессии 60 минут
- Сбор собственного DNS
- Автоматическое открытие заблокированных сайтов
- Приветствие при установке
- GitHub Actions для обновления списка DNS каждые 24 часа

## Установка
1. Скачайте ZIP
2. Распакуйте
3. В Chrome: chrome://extensions/ -> Load unpacked

## Структура
- manifest.json
- popup.html/js
- background.js

## Поисковой робот (GitHub Actions)
См. .github/workflows/update-dns.yml

**Важно:** Расширение не меняет системный DNS напрямую (ограничения браузера), но предоставляет удобный интерфейс и инструкции. Для полного эффекта настройте DNS в ОС.
