# GoidaX - Браузерное расширение для обхода блокировок

Современное расширение на **Node.js** стеке (Vite-ready) для разблокировки YouTube, TikTok и других сайтов.

## Функции
- Большая центральная кнопка активации (60 минут лимит)
- Подмена DNS-рекомендаций (незаблокируемые зарубежные серверы)
- Сбор кастомного DNS для сервисов
- Авто-уведомления при установке/удалении
- Поисковой робот обновляет DNS каждые 24 часа

## Установка
1. Скачай репозиторий
2. Открой `chrome://extensions/`
3. Включи Developer mode
4. Load unpacked → выбери папку расширения

## Разработка
```bash
npm install
npm run build
```

## GitHub Actions
Workflow `update-dns.yml` обновляет `dns-list.json` ежедневно.
